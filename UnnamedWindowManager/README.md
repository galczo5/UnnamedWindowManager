# Unnamed Window Manager

A macOS menu bar app that automatically tiles your windows into a clean, space-efficient layout.

## What it does

When you snap a window, the app arranges it alongside your other snapped windows, splitting the screen into equal tiles. Each new window is added to the layout and everything resizes to fit. You can flip the split orientation between horizontal and vertical at any time.

Windows can be reordered by dragging them onto each other or into edge drop zones. The layout adapts automatically — resize a window manually and the rest adjust to fill the remaining space.

**Auto-snap** watches for new windows and snaps them into the layout automatically. **Auto-organize** repositions all snapped windows whenever the layout changes. Both can be toggled in the config file.

Everything is controlled from a small menu bar icon. You can snap, unsnap, organize, flip orientation, and reload or reset the config — all from the menu.
